# Jogo da velha em java

Projeto simples do clássico jogo da velha, desenvolvido em java. O projeto roda diretamente no terminal, 
permitindo que dois jogadores se alternem para marcar 'X' ou 'O'.

_________________________________________________________________________________________________________

# Funcionalidades: 
* Tabuleiro 3x3;
* Modo 2 jogadores;
* Verificação de vitória;
* Verificação de empate;
 
_________________________________________________________________________________________________________

# Tecnologias utilizadas:
* Linguagem - Java;
* Ambiente - Intellij;
